# Agentic-AI-Digital-Life-System
# Real-time productivity monitoring system using Agentic AI and Streamlit

A real-time productivity monitoring system developed using Agentic AI concepts to help users stay focused on important tasks and reduce digital distractions.

# Project Overview
The system continuously monitors the active application window and intelligently analyzes whether the user is engaged in productive work or entertainment activities. Based on task urgency and user behavior, the system generates smart decisions and real-time alerts to improve productivity.
The project demonstrates how multiple AI agents can collaborate together to perform monitoring, decision-making, task prioritization, and notification handling in real time.

# Features
- Real-time active window monitoring
- Productivity vs entertainment detection
- Intelligent task prioritization
-  Agent-based decision system
- Live Streamlit dashboard
- Activity history tracking
- Productivity analytics graph
- Desktop distraction notifications
- Task completion tracking

# Agents Used
- Behavior Analysis Agent
- Task Management Agent
- Decision Agent
- Action Agent
- Notification Agent
- Analytics Agent
- Monitoring Orchestrator Agent

# Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- PyGetWindow
- Plyer Notifications

# Working Flow
- User enters tasks and deadlines
- System prioritizes tasks automatically
- Active window monitoring starts
- User activity is categorized
- Decision agent evaluates focus level
- Notifications are triggered for distractions
- Dashboard displays live monitoring and analytics

# How to Run:
Install required libraries: pip install streamlit pandas matplotlib pygetwindow plyer
Run the application: streamlit run app.py

# Output
- Live monitoring dashboard
- Activity history table
- Productivity analytics graph
- Real-time alerts and decisions
